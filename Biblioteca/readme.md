# 📚 Library AI Master - Talento Solutions

Sistema avanzado de gestión de bibliotecas desarrollado en Python, asistido por Inteligencia Artificial. Este proyecto demuestra la integración de lógica de negocio compleja, persistencia de datos en SQL y comunicación externa (WhatsApp) dentro de un entorno de escritorio.

## 🚀 Características Principales
- **Gestión Multi-rol**: Interfaz diferenciada para Administradores y Usuarios.
- **Seguridad Robusta**: Almacenamiento de contraseñas mediante hashing SHA-256.
- **Catálogo Visual**: Fichas de libros con portadas cargadas dinámicamente desde URLs.
- **Sistema de Préstamos Inteligente**:
  - Control de stock y límites de usuario.
  - Cálculo automático de fechas de devolución.
  - Sistema de penalizaciones (7 días) por devoluciones tardías.
- **Módulo de Notificación (Mora)**: Detección de libros fuera de plazo y generación de alertas para WhatsApp con un solo clic.
- **Históricos**: Visualización de tablas relacionadas (historial de préstamos por libro).

## 🛠️ Tecnologías
- **Lenguaje**: Python 3.10+
- **Interfaz**: Tkinter (GUI estándar)
- **Base de Datos**: SQLite3 (Local)
- **Procesamiento de Imágenes**: Pillow (PIL)
- **Seguridad**: Hashlib
